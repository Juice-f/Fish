Vänsterklick: Gå
A: Håll åt vänster
D: Håll åt höger
S: Parrera framåtrörelse

---------------------------------------
	How-To-Play
Efter att du har kastat linan så tryck på space när flötet börjar bobba för att börja fånga fisken.
Fånga fiskar genom att fiska. Motverka fiskens rörelser för att trötta ut den. När fisken är helt utmattat så klicka högerklick för att dra in den.

Går linan sönder så gå till tältet och högerklicka för att vila. (Lite finicky men fungerar bäst framför tältet med kameran riktad mot öppningen).

Fångad fisk kan bli upplockad med högerklick. Högerklicka sedan på elden för att grilla och äta den upplockade fisken för att öka din totala stamina för att kunna 
fånga svårare fiskar.

Hjärta: Din stamina. Krok: Linans hållbarhet. Fisk: Fiskens uthållighet.
